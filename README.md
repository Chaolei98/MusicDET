## Abstract

Detecting AI-generated music is crucial for preserving artistic authenticity and preventing the misuse of generative music technologies. However, existing discriminative detectors typically rely on generated samples during training and often suffer from severe performance degradation when confronted with music produced by unseen generators, which limits their real-world applicability. To address this issue, we formulate a zero-shot setting for AI-generated music detection, where the detector is trained exclusively on real music without access to any generated samples. Under this setting, we propose MusicDET, a generator-agnostic detection framework based on frequency-guided normalizing flows that probabilistically models the distribution of real music features. By evaluating the likelihood of an input sample under the learned real-music distribution, MusicDET enables effective detection of out-of-distribution music signals. Experiments on the FakeMusicCaps and SONICS datasets show that MusicDET consistently outperforms conventional discriminative detectors, particularly when detecting music generated by previously unseen models.

## Environment Setup

```bash
conda create -n musicdet python=3.9 -y
conda activate musicdet
cd MusicDET
pip install -r requirements.txt
```

## Dataset Preparation

- Download [FakeMusicCaps](https://zenodo.org/records/15063698) and place the audio files under:

```text
datasets/fakemusiccaps/all_audio_wav/
```

- Download [SONICS](https://github.com/awsaf49/sonics) and place the audio files under:

```text
datasets/sonics/real_songs_wav/
datasets/sonics/fake_songs_wav/
```

- The provided csv files in `label/sonics/` already point to these folders.

### [Optional] Preprocess Audio for Faster Training

You can preprocess downloaded audio in advance for faster and more consistent loading.

`FakeMusicCaps`

```bash
python preprocess_audio.py \
  --in_root /path/to/FakeMusicCaps/all_audio \
  --out_root datasets/fakemusiccaps/all_audio_wav
```

`SONICS`

```bash
python preprocess_audio.py \
  --in_root /path/to/sonics/real_songs \
  --out_root datasets/sonics/real_songs_wav

python preprocess_audio.py \
  --in_root /path/to/sonics/fake_songs \
  --out_root datasets/sonics/fake_songs_wav
```

## Pretrained Models

Download the SSL backbones to `pretrained/`:

```bash
mkdir -p pretrained/xlsr pretrained/mert

huggingface-cli download facebook/wav2vec2-xls-r-300m \
  --local-dir pretrained/xlsr

huggingface-cli download m-a-p/MERT-v1-330M \
  --local-dir pretrained/mert
```

## Training

Train on `FakeMusicCaps`:

```bash
python train.py \
  --task fakemusiccaps \
  --model spec-nf \
  --batch_size 16 \
  --epochs 10 \
  --out_fold output_fakemusiccaps
```

Train on `FakeMusicCaps` with only real samples:

```bash
python train.py \
  --task fakemusiccaps \
  --model spec-nf \
  --batch_size 16 \
  --epochs 10 \
  --out_fold output_fakemusiccaps/only_real \
  --only_real
```

Train on `SONICS`:

```bash
python train.py \
  --task sonics \
  --model spec-nf \
  --batch_size 16 \
  --epochs 10 \
  --out_fold output_sonics
```

## Testing

Test on `FakeMusicCaps`:

```bash
python test.py --task fakemusiccaps --model_path output_fakemusiccaps
```

Test on `SONICS`:

```bash
python test.py --task sonics --model_path output_sonics
```

## Evaluation

```bash
python evaluate --p output_fakemusiccaps
```
